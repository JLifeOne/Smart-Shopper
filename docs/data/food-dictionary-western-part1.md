# Smart Shopper Food Dictionary — Western + Caribbean/Americas Aliases (v0.1 • Part 1)

**Scope**: Western-world representative groceries with Caribbean & the Americas aliases where relevant. Categories are **alphabetical**; items in each category are **alphabetical**. Each entry uses a compact schema:

`Item — category <Category[>Subcategory]>; aliases: [common & regional names]; variants: <forms>; packaging: <typical retail>`

When region-specific names differ, they appear in `aliases` with tags like *(JM)* Jamaica, *(TT)* Trinidad & Tobago, *(PR)* Puerto Rico, *(DO)* Dominican Rep., *(HT)* Haiti (Kreyòl), *(US)* United States, *(CA)* Canada, *(MX)* Mexico, *(BR)* Brazil (PT), *(CO)* Colombia.
